## ğŸ› ï¸ Tech Stack

### Frontend
* **Framework:** Svelte
* **Build Tool:** Vite
* **Styling:** Custom CSS, Toastr (Notifications)
* **Communication:** Socket.io-client

### Backend
* **Runtime:** Node.js
* **Server:** Express.js
* **Database:** SQLite (using `sqlite3` and `sqlite` wrapper)
* **AI Integration:** Groq SDK (Model: `llama-3.3-70b-versatile`)
* **Authentication:** Session-based (express-session) with Bcrypt for password hashing
* **Email:** Nodemailer (configured for Ethereal Email)

## ğŸš€ Getting Started

Follow these instructions to set up the project locally.

### 1. Clone the Repository
```bash
git clone <repository-url>
cd bratz-magazine

```

### 2. Backend Setup

Navigate to the backend directory to install dependencies and configure the database.

```bash
cd backend
npm install

```

**Environment Configuration:**
Create a `.env` file in the `backend/` directory with the following keys:

```env
# AI Service
GROQ_API_KEY=your_groq_api_key_here

# Admin Configuration
ADMIN_PASS=your_secure_password

# Email Service (Mock)
ETH_MAIL=your_ethereal_email
ETH_PASS=your_ethereal_password

```

**Database Initialization:**
Run the setup script to create the SQLite database (`auth.db`) and seed the default admin accounts (Cloe, Jade, Sasha, Yasmin).

```bash
node database/setup.js

```

**Start the Server:**

```bash
node app.js

```

*Server runs on port 8080.*

### 3. Frontend Setup

Open a new terminal window and navigate to the frontend directory.

```bash
cd frontend
npm install
npm run dev

```

*Frontend typically runs on http://localhost:5173.*

### ğŸ”‘ Authentication

2. **Generate Blog:** Click the button to have AI write a post for your character. It will appear in "Pending Drafts."
3. **Publish:** Review a draft and click "Publish" to make it live on the "Corners" and "Home" pages.
4. **Generate Issue:** Click "Publish monthly magazine" to auto-generate a complete issue.
